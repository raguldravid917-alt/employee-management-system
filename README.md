Employee Management System

Tech Stack Used
* **Frontend:** React.js, Tailwind CSS, Vite
* **Backend:** Node.js, Express.js, Multer (for image upload)
* **Database:** MySQL

Features Included
* 100% Figma Pixel Perfect UI (No scrollbars, clean layout).
* View, Add, Edit, and Delete Employee details.
* Profile Image Upload using Multer.
* Prevention of Duplicate Employee ID.

How to Run the Project

1. Database Setup
* Create a MySQL database named `employee_db`.
* Import the provided `employee_db.sql` file into your MySQL database.
